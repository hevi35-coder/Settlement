# ZIP 파일 업로드, 압축 해제 및 비밀번호 검증 기능 개발

**Task ID:** T-003
**Status:** BACKLOG
**Importance:** MUST
**Complexity:** 6/10
**Urgency:** 8/10
**Dependencies:** T-002

## Description

### 요구사항
- ZIP 파일을 multipart/form-data로 업로드
- 환경변수에 저장된 비밀번호로 압축 해제
- 비밀번호 오류 및 형식 오류 처리
### 구현 세부사항
- Next.js API Route(/api/upload) 내 Vercel Serverless Function 작성
- jszip 라이브러리로 메모리상 ZIP 파일 로드 및 ZIP_PASSWORD 환경변수 사용하여 압축 해제
- xlsx 라이브러리로 .xlsx 파일 탐색 전 준비
- 악성 파일 또는 형식 불일치 시 400 오류 반환
### 테스트 전략
- 단위 테스트: jszip 모킹으로 정상/비밀번호 오류 시나리오 검증
- 통합 테스트: API Route에 가짜 ZIP 파일 전송 후 응답 상태·메시지 검증
- 예외 테스트: 지원하지 않는 파일 형식 및 잘못된 비밀번호 업로드 테스트
