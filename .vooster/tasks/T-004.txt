# 엑셀 파싱 및 '가계부 내역' 시트 데이터 추출 로직 구현

**Task ID:** T-004
**Status:** DONE
**Importance:** MUST
**Complexity:** 7/10
**Urgency:** 9/10
**Dependencies:** T-003

## Description

### 요구사항
- 압축 해제된 .xlsx 파일에서 '가계부 내역' 시트 추출
- 지정된 기간(startDate, endDate) 내 지출 데이터(날짜, 타입, 대분류, 소분류, 내용, 금액, 메모, 결제수단) 파싱
### 구현 세부사항
- xlsx 라이브러리 이용하여 워크북 로드 및 SheetNames에서 '가계부 내역' 선택
- 각 행 row 데이터 매핑 후 JSON 객체 배열로 변환
- 유효하지 않은 데이터(날짜 형식, 필수 열 누락) 시 예외 처리
### 테스트 전략
- 단위 테스트: 다양한 엑셀 샘플 파일로 파싱 결과 일치 여부 검증
- 경계 테스트: 기간 외 데이터 필터링 기능 확인
- 에러 테스트: 시트명 누락 또는 필수 컬럼 누락 시 에러 핸들링 검증
